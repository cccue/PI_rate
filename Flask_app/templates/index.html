Ahoy!: Pi-rate is a little project conceived by a mathematician (me) who got involved in collecting, analyzing and visualizing rum review-data while having some fun learning (a bit of) web technologies. The project involved web scraping of the RumRatings site employing a combination of Beautiful Soup and Dryscrape in order to collect the data. Processing, analysis and visualization were achieved via Python and Javascript integrated into Flask as the backend tool chosen to build this simple web app.

Check the "Analytics" dropdown blue menu at the top of the page for examples on analyzing and visualizing the rum data in alternative ways to those found at RumRatings.
